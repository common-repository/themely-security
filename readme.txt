== Description ==

WordPress Security, Malware Prevention, Anti-spam & Firewall. Super simple, install-and-forget, WordPress security plugin for everyone.

= FEATURES INCLUDE =

- Limits login attempts
- Disables pingbacks
- Disables access to xmlrpc.php
- Protects wp-config.php
- Disables directory browsing
- Blocks XSS attacks
- Blocks wp-includes files & folders
- Hides version strings
- Deploys 6G firewall and blacklist https://perishablepress.com/6g/#6g-firewall
- Removes readme.html file
- Math captcha for login, registration and comment forms
- Automatic updates for WordPress core, plugins and themes
- Removes Website(URL) field from comments form

= HOW TO STOP ALL COMMENT SPAM WITHOUT A PLUGIN =

This article by Jeff Star outlines existing settings in WordPress to completely eliminate or significantly reduce comment spam without the help of a plugin https://digwp.com/2009/11/dont-need-plugins-to-stop-comment-spam/

- Comment author must fill out name and e-mail
- Users must be registered and logged in to comment
- Automatically close comments on articles older than X days
- Break comments into pages with X comments per page
- Before a comment appears an administrator must always approve the comment
- Comment author must have a previously approved comment
- Hold a comment in the queue if it contains X or more links
- Add the following words **href=**, **https://** and **http://** to the Comment Blocklist field to trash any comment containing links in the comment field

== Installation ==

1. Log into your Admin Dashboard
2. Navigate to the Plugins page.
3. Click on the Install button.
4. Upload this plugin.
5. Once installed, activate the plugin.
